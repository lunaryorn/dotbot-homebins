# dotbot-gsettings

A [dotbot] plugin to install tools with [homebins].

```yaml
- homebins:
    # Update all installed binaries
    update: true
    # Install binaries
    add:
      - bat
      - exa
      - fd
      - git-gone
      - hub
      - jq
```

[dotbot]: https://github.com/anishathalye/dotbot
[homebins]: https://github.com/lunaryorn/homebins
